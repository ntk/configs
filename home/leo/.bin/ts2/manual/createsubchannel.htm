Create Sub-Channel

 

This functions allows users to create sub-channels. You must give your subchannel a name. Depending on the server setting (under Menu Connection, Server Connection Info under Server Options or right-click on the server name and choose Server Connection Info in the popup menu, under Server Options) only certain persons can create sub-channels or everyone can (if “all can create channels” is specified in the by the admin when creating the server). You can set a topic for your subchannel in the Topic field. In the Description field you can add message for the people that join the sub-channel. If you want to restrict the number of people that are allowed to join your sub-channel set it in the Max. Users box. The subchannel inherits all the flags and the password of the parent channel, that's why the boxes are greyed out. When you are done press the Create Channel button to make the sub-channel or Cancel to cancel the sub-channel. Server Admins are allowed to create sub-channels everywhere. Channel Admins can create sub-channels in their own channels.
