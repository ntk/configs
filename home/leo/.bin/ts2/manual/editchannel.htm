Edit Channel

 

Basically this function lets you edit an existing channel. It is the same as the Create channel function with the exception of one thing. It has an “order” box (next to “Max Users”) which allows you to order channels. If a channel is given the value of 0 it is on top of the channel list in a server. Channels with the same value (for example the default value of 32000) are sorted alphabetically.
