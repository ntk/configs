How it works

 

Basically you start up your client, fill in an ip or DNS name in the the Connection --> Connect menu and connect to a server someone has setup earlier. If all works well you should be good to go. It has numerous functions and features which set it apart from other voice communication programs that are like TeamSpeak. The basic idea is an IRC layout with resembling features. This is just a very brief description to give a really short overview of the application.
