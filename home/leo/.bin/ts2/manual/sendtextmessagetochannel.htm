Send Text Message to Channel

 

You can send text messages to a channel (a nice feature when for example complicated URL’s need to be passed on). First click on the channel you want to send a message to, otherwise you will send a message to the channel that is highlighted or the channel you are currently in. Then click on Send Text to Channel. A new window appears with the channel you are sending your message to in the top bar. You can type messages (up to 255 characters) in the Message field. If you want to display your message in another color than the standard black, you can modify it in the Color box. When you’re done click the Ok button to send the message to the channel you chose or cancel to cancel the message.

Server Admins are allowed to send text messages to all channels (wether they are moderated or not), Channel Admins to the channels they are admin over. In unmoderated channels everybody can send messages (provided they are registered).
