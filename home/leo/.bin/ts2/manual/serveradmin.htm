Server Admin

 

This function allows the highlighted person to become Server Admin. A Server Admin can do ALL things available in TeamSpeak 2. He can give and take away all statuses to everyone (including himself!). Server Admins can create,edit and delete all channels and get full information about all the persons in the server. They can send text to all channels. They have all kick functions at their disposal too and can give persons the right to register with the server (if the server hasn’t got the Allow all to register server-flag). When given to a registered user this user will become a permanent SA as long as he logs in with his Login name and password. When given to an unregistered user it will give him SA as long as he is connected to the server.
