What do I need for TS?

 

For the Windows client you need at least a Pentium 166mhz computer with 64 mb of ram. The program itself only needs 5 mb of HD space. The linux client needs 15 mb of HD space. You to have at least Windows 98 in order to run the client. For the Windows server you need a Pentium class computer with 64 mb of ram. It needs to run Windows 98 or higher. You need to have DirectX 8.1 installed! For the linux server you need a Pentium 133 mhz computer with 32 mb of memory. Depending on the number of people that will connect and which codecs are used, the server needs a good internet connection. When you are using the GSM codecs you need at least a DSL connection or better, depending on the number of people connected. In the section Used codecs and some words about quality things will be explained further.
